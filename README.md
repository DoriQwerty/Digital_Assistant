# Digital Assistant

## Описание проекта

Digital Assistant — это веб-приложение для обработки аудиозаписей встреч и разговоров. Система автоматически преобразует речь в текст, выделяет ключевые моменты беседы, определяет участников и формирует список задач по итогам разговора.

Для распознавания речи используется модель Whisper.

## Возможности

* загрузка аудиофайлов;
* автоматическая расшифровка речи;
* получение краткого содержания разговора;
* выделение участников беседы;
* формирование списка задач;
* работа через простой веб-интерфейс.

## Используемые технологии

* Python
* FastAPI
* OpenAI Whisper
* HTML
* CSS
* JavaScript

## Установка

1. Склонировать репозиторий или скачать архив проекта.

2. Создать виртуальное окружение:

```bash
python -m venv venv
```

3. Активировать виртуальное окружение:

Windows:

```bash
venv\Scripts\activate
```

4. Установить необходимые библиотеки:

```bash
pip install -r requirements.txt
```

5. Запустить сервер:

```bash
uvicorn server:app --reload
```

После запуска сервер будет доступен по адресу:

```text
http://127.0.0.1:8000
```

## Использование

1. Запустить сервер.
2. Открыть файл `index.html`.
3. Выбрать аудиофайл.
4. Нажать кнопку обработки.
5. Дождаться завершения анализа.

В результате будут отображены:

* полный текст разговора;
* краткое содержание;
* список участников;
* найденные задачи.

## Итог

Разработанное приложение позволяет автоматизировать обработку аудиозаписей и ускоряет получение основной информации из разговоров и встреч.
